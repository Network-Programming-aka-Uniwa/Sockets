# INSTALL

## Sockets

This guide explains how to set up, build, and run the project on your local machine.

---

## 1. Requirements

- **Java Development Kit (JDK):** Version 8 or higher
- **Operating System:** Linux / Windows / MacOS
- **Development Environment:** Command Line or any Java IDE (Eclipse, IntelliJ IDEA, NetBeans)

---

## 2. Installation & Usage

### 2.1 Clone the Repository

```bash
git clone https://github.com/Network-Programming-aka-Uniwa/Sockets.git
cd Sockets
```

### 2.2 Compile the Java Files

```bash
javac Server.java Client.java
```

### 2.3 Run the Server

```bash
java Server
```

### 2.4 Run the Client

```bash
java Client
```
